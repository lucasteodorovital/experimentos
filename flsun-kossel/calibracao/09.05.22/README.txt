#Nota:
* Após ajustar o steps/mm, sempre calibrar o delta height. Os parâmetros atuais podem ser consultados com m503 (EEPROM).
* M92: Ajusta steps/mm
* M665: Ajuste de delta parameters (radius/height/diagonal-rod)

#Características da amostra *antes* dos ajustes:
* Presença sutil de *layer shift*;
* *Blobs* bem evidentes;
* *Poor layer adhesion*;
* Primeira camada ruim;
* Dimensões incorretas (menores) para todos os eixos.
